## 0.0.1

- Initial release.
- Interactive map powered by `flutter_map` and OpenStreetMap tiles.
- Address search via the Nominatim geocoding API (no API key required).
- One-tap GPS / current-location button.
- Full theming via `LocationPickerTheme`.
- Built-in Arabic and English UI strings with `LocationPickerStrings`.
- BLoC/Cubit state management — no global state pollution.
- Supports Android, iOS, Web, Windows, macOS, and Linux.
